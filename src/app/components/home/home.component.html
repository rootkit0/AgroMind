🚀 Angular 20 + Firebase Auth Starter Template

This project is a base template built with Angular 20, designed to provide a modern, secure, and efficient solution for applications requiring user authentication. It comes fully configured with Firebase Authentication, standalone components, custom guards, and the latest Angular improvements.

✨ Features

✅ Angular 20 with standalone components support

🔐 Full integration with Firebase Auth (email/password, Google, etc.)

🔄 Custom auth guards to protect routes

🚀 Centralized `AuthService` to manage user state

🧱 Ready to scale and extend with new features
